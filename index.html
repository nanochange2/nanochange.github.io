The process

At Biotein, we have a simple process.

1. We take fluorescence microscopy pictures of the mitochondria of pure senescent cells, live cells, and dead cell and form a dataset using those pictures.

2. We run the dataset through a convolutional neural network and find key features of the cells (biomarkers and markers).

3. We find targeted therapies using those biomarkers and markers we've found.
